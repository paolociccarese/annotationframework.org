UIMA integration through Clerezza

UIMA is an OASIS standard that allows the definition of analysis pipelines to manage unstructured information and extract structures and semantics around given data. The Clerezza-UIMA integration brings the power of UIMA inside Clerezza allowing reuse of existing UIMA components in a Linked Open Data compliant manner.Clerezza can transform the extraction results in Annotation Ontology (AO) format so that they can be imported by Domeo.
